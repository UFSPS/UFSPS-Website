This page collects the chapter resources that help members move faster: repositories, talks, community spaces, and reference links that support technical growth between meetings.

How to use this page

Start with the resources closest to your next step.

  • If you are new, begin with the community links and the workshop repository.
  • If you want to understand the chapter tone, watch the public talk or project summaries.
  • If you want broader context, use the IEEE SPS reference links and partner sites.
